# c17-bad-project-10-tw

### SETUP

1. yarn init

```bash
yarn init -y
```

2. Install TypeScript

```bash
yarn add --dev typescript ts-jest @types/jest @types/node ts-node ts-node-dev
```

3. Install express

```bash
yarn add express @types/express
```

4. Install knex

```bash
yarn add knex @types/knex
```

5. Install jest

```bash
yarn add --dev jest
yarn ts-jest config:init
```
